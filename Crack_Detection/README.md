# [Sub-Module] Surface Crack Detection

##  Context in FYP
This module is developed to identify structural fractures on surfaces using Deep Learning. It complements the LiDAR and Thermal modules in our multi-sensor suite.

##  Implementation
- **Source:** Developed in Google Colab.
- **Goal:** Real-time identification of concrete and metal cracks.

##  Folder Contents
- `Crack_Detection_Model.ipynb`: The full training and testing code (Click "Download" if the preview doesn't load).
- `best.pt`:Trained weights for the crack model.
